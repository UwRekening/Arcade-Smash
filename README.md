## Controls?

<h2>Je hebt een Super Nintendo Entertainment System-controller nodig om dit te spelen!</h2>

<p>Arrows - Movement</p>
<p>X - Spring naar boven</p>
<p>B - Spring naar beneden</p>
<p>A - Aanvallen</p>
<p>L - Repareren</p>
<p>R - Kapot maken</p>
